voting app functionality :

1. user signup / signin
2. see list of candidates
3. one vote per user , after voting users can't vote again
4. there is a route which will show list of candidates and their live vote count (sorted by their vote count)
5. user data must contain their unique government id proof named : aadhar card number
6. there should be one admin who will maintain all the candidate list and make sure that he can't cast his vote 
7. users can change their password
8. users can login uisng aadhar number and password

Routes :

User Authentication:

/signup : POST - create a new user account
/login : POST - Log in to existing account   [aadhar card number + password]

Voting:

/candidates : GET - get the list of candidates 
/vote/:candidateId : POST - vote a specific candidate 

Vote Counts:

/vote/counts : GET - get the list of candidate sorted by their vote count 


User Profile:

/profile : GET - Get the users profile infirmation
/profile/password - PUT - change the user's password


Admin Candidate Management:

/candidates: POST - Create a new Candidate
/candidates/:candidateId: PUT - update the existing candidate 
/candidates/:candidateId: DELETE - Delete a candidate from the list
